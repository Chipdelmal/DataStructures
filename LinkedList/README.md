# Linked List

## Data Structure Summary

* Data is stored in non contiguous memory locations
* No arbitrary access to data (sequential only)

Useful for: variable lengths, homogeneous data types

### Complexity

| Action  |  Cost |
|---------|-------|
| Access  | O(n)  |
| Search  | O(n)  |
| Insert (beginning)  | O(1) |
| Insert (end)  | O(n) |
